# Sistema de Inventario

CRUD de productos con Node.js, Express y SQLite.

## Uso

1. Ejecuta `npm install`
2. Ejecuta `npm start`
3. Abre `http://localhost:3007`

## Funcionalidades

- Listar productos
- Crear producto
- Editar producto
- Eliminar producto

## Tecnologías

- Node.js
- Express
- SQLite
- HTML, CSS y JavaScript
